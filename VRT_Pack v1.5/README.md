# VRT_Pack

VRT服务器专用数据包

# 功能

## 彩色名字

之前是个计分榜显示，后采用mcdr的statsHelper，因此计分榜显示被删除，切换彩色名字的功能作为特色保留

## 血量显示

在玩家列表和玩家头顶名字显示当前玩家血量

## 附魔金苹果合成

将8/64个金块和1/8个苹果丢在地上，手动合成1/8个盗版附魔金苹果

## 席地而坐

I wannna sit EVERYWHERE!!!
